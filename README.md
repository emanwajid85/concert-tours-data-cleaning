# Concert Tour Data Cleaning

Cleaned a real-world messy dataset of concert tour earnings using Python and pandas.

## What this project does
- Removes columns with excessive missing data
- Cleans hidden Unicode characters and footnote markers from text fields
- Converts currency columns from text to numeric values
- Sorts and exports a clean, analysis-ready dataset

## Tools used
Python, pandas

## Files
- `project1.py` — cleaning script
- Raw and cleaned CSV files included
